<h3 align="center"> O Role20 é um projeto criado para catalogar lanchonetes baratas, exibindo estabelecimentos que oferecem produtos por até R$29. A aplicação combina Django no backend e React no frontend, com deploy realizado na Railway. </h3> <hr>

# APRENDIZADOS:
Durante o desenvolvimento, dois aprendizados foram fundamentais:

1. Resolução do problema N+1 no Django
A API inicialmente executava múltiplas consultas redundantes ao banco. A aplicação correta de select_related e prefetch_related eliminou o problema N+1, resultando em respostas mais rápidas e melhor uso dos recursos.

2. Uso do TanStack Query para melhorar a experiência do usuário
No frontend, o TanStack Query foi utilizado para cache, sincronização e gerenciamento eficiente dos dados da API. Isso reduziu o número de requisições, acelerou carregamentos e tornou a navegação mais fluida.

O projeto reforça práticas de otimização fullstack, com backend performático e frontend responsivo, servindo como base para futuras melhorias e expansão da ideia.
